#  Online Food Delivery System

A Java EE based Online Food Delivery Web Application developed using JSP, Servlets, JDBC, MySQL, HTML, CSS, JavaScript, and Apache Tomcat.

##  Features

- User Registration
- User Login
- Restaurant Listing
- Food Menu
- Add to Cart
- Checkout
- Order Confirmation

##  Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat
- Eclipse IDE
- Maven

##  Project Structure

```
OnlineFoodDeliverySystem
│── src/
│── target/
│── pom.xml
│── .project
│── .classpath
```

##  How to Run

1. Clone the repository.
2. Import the project into Eclipse.
3. Configure Apache Tomcat.
4. Configure MySQL Database.
5. Run the project on Tomcat Server.
